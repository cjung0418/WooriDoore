# Woori Doore


## **프로젝트 소개**

`WOORIDOORE`는 기존의 각종 플랫폼을 통해 진행되었던 공동구매, 소규모 펀딩에서의 여러 문제를 해결하기 위해 탄생했습니다. 펀딩 주최자 및 펀딩 플랫폼에 대한 신뢰성을 확보하기 위한 블록체인을 사용한 펀딩 플랫폼 우리두레, 이제 믿고 투자해보세요.
